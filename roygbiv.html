roygbiv

poster series | 18” × 24” | 2017/2022

posters made for a movie festival called roygbiv. roygbiv is a movie celebration highlighting the joy and fun a movie can have with color. every movie showing displays one main color starting with the color red on day 1 and ending the festival with the color purple on day 3.


by using only typography and color, each poster contains tiny fragments that make up the word “roygbiv”. each fragment echoes more color the same way each movie showing does in their creative individual way. ultimately by placing every poster on top of each other, they slowly create the word “roygbiv”—showing a wide range of colors identical to how it would look if you layered every movie shown in the festival on top of each other.
